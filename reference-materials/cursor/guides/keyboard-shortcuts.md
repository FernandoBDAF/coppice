## Navigation

### Basic Navigation

```
Cmd/Ctrl + P: Quick file open
Cmd/Ctrl + Shift + P: Command palette
Cmd/Ctrl + B: Toggle sidebar
Cmd/Ctrl + J: Toggle terminal
Cmd/Ctrl + Shift + E: Explorer view
Cmd/Ctrl + Shift + F: Search view
Cmd/Ctrl + Shift + G: Source control view
Cmd/Ctrl + Shift + D: Debug view
Cmd/Ctrl + Shift + X: Extensions view
```

### Editor Navigation

```
Cmd/Ctrl + G: Go to line
Cmd/Ctrl + Shift + O: Go to symbol
Cmd/Ctrl + T: Go to symbol in workspace
Cmd/Ctrl + Shift + M: Show problems
Cmd/Ctrl + Shift + Tab: Navigate editor history
Cmd/Ctrl + Tab: Navigate editor history (forward)
```

### Window Management

```
Cmd/Ctrl + W: Close editor
Cmd/Ctrl + Shift + W: Close all editors
Cmd/Ctrl + K F: Close folder
Cmd/Ctrl + K R: Reveal active file in explorer
Cmd/Ctrl + K O: Open active file in new window
```

## Editing

### Basic Editing

```
Cmd/Ctrl + X: Cut line
Cmd/Ctrl + C: Copy line
Cmd/Ctrl + V: Paste
Cmd/Ctrl + Shift + K: Delete line
Cmd/Ctrl + Enter: Insert line below
Cmd/Ctrl + Shift + Enter: Insert line above
Cmd/Ctrl + Shift + \: Jump to matching bracket
Cmd/Ctrl + /: Toggle comment
Cmd/Ctrl + U: Undo cursor operation
Cmd/Ctrl + Shift + U: Redo cursor operation
```

### Code Manipulation

```
Cmd/Ctrl + Shift + [: Fold code block
Cmd/Ctrl + Shift + ]: Unfold code block
Cmd/Ctrl + K Cmd/Ctrl + [: Fold all
Cmd/Ctrl + K Cmd/Ctrl + ]: Unfold all
Cmd/Ctrl + K Cmd/Ctrl + 0: Fold all
Cmd/Ctrl + K Cmd/Ctrl + J: Unfold all
```

### Indentation

```
Tab: Indent line
Shift + Tab: Outdent line
Cmd/Ctrl + [: Indent line
Cmd/Ctrl + ]: Outdent line
Cmd/Ctrl + K Cmd/Ctrl + [: Indent region
Cmd/Ctrl + K Cmd/Ctrl + ]: Outdent region
```

## Search and Replace

### Search

```
Cmd/Ctrl + F: Find
Cmd/Ctrl + H: Replace
Cmd/Ctrl + Shift + F: Find in files
Cmd/Ctrl + Shift + H: Replace in files
Cmd/Ctrl + D: Select next occurrence
Cmd/Ctrl + K Cmd/Ctrl + D: Skip selection
Cmd/Ctrl + U: Undo last selection
```

### Replace

```
Cmd/Ctrl + H: Replace
Cmd/Ctrl + Shift + H: Replace in files
Cmd/Ctrl + Alt + Enter: Replace all
Cmd/Ctrl + Shift + 1: Replace in selection
```

## Multi-Cursor and Selection

### Multi-Cursor

```
Cmd/Ctrl + D: Add selection to next find match
Cmd/Ctrl + K Cmd/Ctrl + D: Move last selection to next find match
Cmd/Ctrl + U: Undo last cursor operation
Cmd/Ctrl + Shift + L: Select all occurrences of current selection
Cmd/Ctrl + Alt + Up/Down: Add cursor above/below
Cmd/Ctrl + Shift + Alt + Up/Down: Add cursor above/below
```

### Selection

```
Cmd/Ctrl + A: Select all
Cmd/Ctrl + Shift + L: Select all occurrences
Cmd/Ctrl + F2: Select all occurrences of word
Cmd/Ctrl + Shift + Space: Select current scope
Cmd/Ctrl + K Cmd/Ctrl + Space: Select current scope
```

## Productivity Tips

### General Tips

1. Use command palette (Cmd/Ctrl + Shift + P) for quick access to all commands
2. Customize keyboard shortcuts to match your workflow
3. Use multi-cursor editing for repetitive tasks
4. Leverage AI features for code generation and review
5. Use code folding to manage large files

### Code Navigation

1. Use "Go to Symbol" (Cmd/Ctrl + Shift + O) to quickly find functions and classes
2. Use "Go to Line" (Cmd/Ctrl + G) for precise navigation
3. Use "Go to File" (Cmd/Ctrl + P) for quick file access
4. Use "Go to Definition" (F12) to navigate to symbol definitions

### Editing Efficiency

1. Use multi-cursor editing for parallel changes
2. Use code folding to focus on relevant sections
3. Use AI suggestions for code completion
4. Use refactoring tools for code improvements

### AI Integration

1. Use AI chat for quick questions and answers
2. Use code generation for repetitive tasks
3. Use code review for quality assurance
4. Use AI suggestions for code improvements
