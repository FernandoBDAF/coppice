## Defects the experiment-building itself caught (the method works)

1. MIME allow-list unreachable for text (`text/plain; charset=utf-8` vs bare
   type) — text/markdown uploads could never succeed.
2. document.process payload violated MESSAGE_FORMAT.md (`bucket` vs
   `storage_bucket`) — graphrag rejected every real document task.
3. api migration 000002 not idempotent — compose one-shot re-runs failed.
4. Finding (not yet fixed, by design → v4/ADR-008.3): document status is
   write-only; no worker→API results path exists.
